Batch File Renaming Tool

Greetings, fair file namer, may this page find you well and productive.

To use this tool, select the files you wish to rename in sequence, enter the name that you wish to assign to your files, and use '{n}' after.

For example - MGP APP 24_0{n} will produce MGP APP 24_01, MGP APP 24_02, ect...
